## ūüďĖ About This Project

**LinNeo** is a biodiversity knowledge graph containing **2.8+ million species** with their complete taxonomic hierarchies and geographic distributions. This project was created as a research tool for my book **[100%](https://github.com/RemiH06/100)** (might be private for now), helping me understand real-world biodiversity patterns to create more authentic invented species and ecosystems.
